The goal of instmplts is to facilitate the setup of for the report in
the UStar2-course

## Installation

You can install the version from [GitHub](https://github.com/) with help
of the {remotes}-package:

``` r
# install.packages("remotes")
remotes::install_github("cathblatter/instmplts")
```

## How to use it?

After installing the package you will have access to the
Rmarkdown-Templates.

1.  Open a new .Rmd-file *File –&gt; New File –&gt; R Markdown*

2.  Under “From Template” select the “Report UStaR2”. You need to choose
    the directory to place the report.
